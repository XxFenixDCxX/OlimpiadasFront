01/03/2024
Periodo de Registro de compradores

Durante este período, los usuarios pueden registrarse para participar en el sorteo de entradas. La interfaz pública guía a los usuarios a través del proceso de registro y proporciona información relevante sobre la etapa actual del proceso de compra.

Comienza este proceso Registrandote desde el boton de abajo o visualiza el estado actual del proceso iniciando sesion desde el mismo boton.

30/03/2024
Sorteo de Entradas

En este día se lleva a cabo el sorteo para asignar a los compradores registrados a uno de los 4 slots de compra. Cada slot representa una semana en la cual los compradores pueden adquirir entradas para eventos específicos.

01/04/2024
Periodo 1 de Compra entradas

Durante este período, los compradores asignados por sorteo tienen la oportunidad de adquirir entradas para eventos específicos. Cada semana, se permite la compra de un máximo de 5 entradas para un máximo de 3 eventos.

01/05/2024
Periodo 2 de Compra de Entradas

Durante este período se realizan las compras de las entradas asignadas para los eventos del segundo periodo, así como las entradas sobrantes del primer periodo. Los usuarios pueden adquirir sus entradas a través de la interfaz web o móvil proporcionada.
